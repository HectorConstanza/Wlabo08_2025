¿Cuál es la diferencia entre autenticación y autorizacion?
Autenticación: verifica quién eres (ej. iniciar sesión con usuario y contraseña) y Autorización: verifica qué puedes hacer (ej. acceder a rutas protegidas según tu rol).
¿Cuál es la función del token JWT en la guía?
Se genera al iniciar sesión.
Se envía en cada solicitud para acceder a rutas protegidas.
Permite validar identidad y controlar acceso sin guardar sesiones en el servidor.
